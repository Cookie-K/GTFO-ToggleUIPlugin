# Toggle UI Plugin

A simple plugin to toggle UI and body visibility 

## Default Key Bind Config
- ToggleUIKey: F1
- ToggleBodyKey: F2

## API for Plugin Devs
For plugin developers that want control over UI and body visibility the following methods are exposed by `ToggleUIManager`

- `ToggleUI`
- `ToggleBody`
- `HideUI`
- `ShowUI`
- `HideBody`
- `ShowBody`
