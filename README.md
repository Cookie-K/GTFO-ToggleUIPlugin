# Toggle UI Plugin

A simple plugin to toggle UI and body visibility 

## Default Key Bind Config
- Toggle UI Key: F1
- Toggle Body Key: F2
- Exclude Interaction UI: true (if set to false hiding the UI will also hide any interaction prompts or security scan progress)
